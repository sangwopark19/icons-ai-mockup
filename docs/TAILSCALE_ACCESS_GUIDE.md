## 🤔 왜 Tailscale이 필요한가요?

### 문제 상황

MockupAI 서버는 회사 내부 Mac 컴퓨터에서 운영되고 있습니다. 하지만 **회사 네트워크 구조상 몇 가지 문제**가 있었습니다:

| 문제 | 설명 |
|------|------|
| **부서별 다른 네트워크** | 각 부서마다 IP 대역이 달라서 서로 접속이 안 됨 |
| **보안 위험** | 모든 곳에서 접속 가능하게 열면 해커 공격에 취약 |
| **외부 접속 불가** | 재택근무나 외부 출장 시 사용 불가 |

### 해결책: Tailscale VPN

며칠간의 연구 끝에 **Tailscale**이라는 VPN 솔루션을 도입했습니다.

**Tailscale이란?**
- 전 세계 어디서든 같은 네트워크에 있는 것처럼 연결해주는 **가상 사설 네트워크(VPN)**
- 일반 VPN과 달리 **설치가 매우 간단**하고 **속도가 빠름**
- 군용 수준의 **강력한 암호화** (WireGuard 기반)
- **무료**로 100명까지 사용 가능

### Tailscale의 장점

| 장점 | 설명 |
|------|------|
| **보안** | 허가된 사람만 접속 가능, 외부 해커 차단 |
| **편리함** | 한 번 설치하면 자동 연결 |
| **어디서든** | 회사, 집, 카페, 해외 어디서든 접속 가능 |
| **빠른 속도** | 직접 연결 방식으로 속도 저하 최소화 |

### 왜 조금 귀찮아도 설치해야 하나요?

> "그냥 바로 접속하면 안 되나요?"

**안 됩니다.** 그 이유는:

1. **해커 공격 위험**: 서버를 인터넷에 완전히 공개하면 24시간 내에 수천 건의 해킹 시도가 들어옵니다
2. **회사 데이터 보호**: 우리가 만드는 제품 디자인 이미지는 중요한 회사 자산입니다
3. **AI API 비용**: 해커가 접속하면 비싼 AI API 비용이 무단으로 사용될 수 있습니다

**5분만 투자해서 Tailscale을 설치하면, 이후로는 아무 걱정 없이 안전하게 사용할 수 있습니다!**

## 💻 Tailscale 설치하기

### Windows 사용자

#### 1단계: Tailscale 다운로드

1. 아래 링크를 클릭하여 다운로드 페이지로 이동:
   
   **https://tailscale.com/download/windows**

2. **"Download Tailscale for Windows"** 버튼 클릭

3. `tailscale-setup-x.xx.x.exe` 파일이 다운로드됩니다

#### 2단계: 설치하기

1. 다운로드된 **설치 파일을 더블클릭**하여 실행

2. 설치 마법사가 나타나면 **"Next"** → **"Install"** 클릭

3. 설치가 완료되면 **"Finish"** 클릭

#### 3단계: 로그인하기

1. 화면 **오른쪽 하단 작업표시줄**에서 Tailscale 아이콘을 찾습니다
   - 아이콘이 안 보이면 **↑ 화살표** 클릭하여 숨겨진 아이콘 확인

2. Tailscale 아이콘을 **우클릭** → **"Log in..."** 클릭

3. 웹 브라우저가 열리면 로그인 화면이 나타납니다

4. **"Sign in with Email"** 또는 해당 로그인 방식 선택

5. 위 [접속 정보](#-접속-정보)에 있는 **이메일과 비밀번호** 입력

6. 로그인 완료! 이제 Tailscale이 연결되었습니다

#### 4단계: 연결 확인

1. 작업표시줄의 Tailscale 아이콘이 **활성화 상태**인지 확인
   - 아이콘에 녹색 체크가 있으면 정상

2. 아이콘 우클릭 → 연결된 기기 목록에 **MockupAI 서버**가 보이면 성공!

---

### Mac 사용자

#### 1단계: Tailscale 다운로드

**방법 A: 공식 사이트에서 다운로드 (권장)**

1. 아래 링크 클릭:
   
   **https://pkgs.tailscale.com/stable/#macos**

2. **"Download Tailscale for macOS"** 클릭

3. `.zip` 파일 다운로드 후 압축 해제

4. `Tailscale.app`을 **응용 프로그램** 폴더로 드래그

**방법 B: Mac App Store에서 설치**

1. Mac App Store 열기
2. "Tailscale" 검색
3. **"받기"** 클릭하여 설치

#### 2단계: 실행 및 로그인

1. **응용 프로그램** 폴더에서 **Tailscale** 실행

2. 처음 실행 시 **"시스템 확장 프로그램"** 허용 요청이 나타납니다
   - **"허용"** 클릭
   - 필요시 **시스템 설정 → 개인정보 보호 및 보안**에서 허용

3. 상단 메뉴바에 **Tailscale 아이콘**이 나타납니다

4. 아이콘 클릭 → **"Log in..."** 클릭

5. 웹 브라우저에서 로그인 화면이 열립니다

6. **"Sign in with Email"** 선택

7. 위 [접속 정보](#-접속-정보)의 **이메일과 비밀번호** 입력

8. **"Connect"** 또는 **"Authorize"** 클릭

9. 연결 완료!

#### 3단계: 연결 확인

1. 상단 메뉴바의 Tailscale 아이콘 클릭

2. **"Connected"** 상태 확인

3. 연결된 기기 목록에 MockupAI 서버가 보이면 성공!

## 🌐 MockupAI 접속하기

Tailscale 설치 및 로그인이 완료되었다면:

### 1. 웹 브라우저 열기

Chrome, Edge, Safari 등 아무 브라우저나 사용 가능

### 2. 주소 입력

주소창에 아래 주소 입력:

```
http://100.69.75.47:3000/
```



### 3. 회원가입 / 로그인

- 처음 사용자: **회원가입** 후 사용
- 기존 사용자: MockupAI 계정으로 **로그인**

> 💡 **참고**: MockupAI 로그인 계정은 Tailscale 계정과 **별개**입니다!
> - Tailscale 계정: 네트워크 접속용 (공용)
> - MockupAI 계정: 앱 사용자 계정 (개인별)

### 4. 사용 시작!

이제 MockupAI의 모든 기능을 사용할 수 있습니다:
- IP 변경
- 스케치 실사화
- 히스토리 관리
- 고해상도 다운로드

자세한 사용법은 **[사용자 가이드](USER_GUIDE.md)** 를 참조하세요.

## ❓ 자주 묻는 질문

### Q1. Tailscale을 항상 켜놔야 하나요?

**A**: 네, MockupAI를 사용하는 동안에는 Tailscale이 연결되어 있어야 합니다. 
- 다행히 Tailscale은 **자동 실행** 설정이 되어 있어 컴퓨터를 켜면 자동으로 연결됩니다
- 배터리 소모나 인터넷 속도에는 거의 영향이 없습니다

### Q2. 집에서도 접속할 수 있나요?

**A**: 네! Tailscale의 장점이 바로 **어디서든 접속 가능**하다는 것입니다.
- 집, 카페, 해외 출장 중에도 사용 가능
- Tailscale만 연결되어 있으면 됩니다

### Q3. Tailscale이 유료인가요?

**A**: 우리가 사용하는 플랜은 **무료**입니다.
- 개인 플랜: 무료로 100대 기기까지 사용 가능
- 회사에서 비용 부담 없이 사용 중입니다

### Q4. 보안상 안전한가요?

**A**: 매우 안전합니다.
- **WireGuard** 기반 군용 수준 암호화
- 모든 통신이 **종단간 암호화**됨
- 로그인한 사람만 접속 가능

### Q5. 설치했는데 접속이 안 돼요

**A**: 아래 체크리스트를 확인하세요:
1. ✅ Tailscale 아이콘이 활성화 상태인가?
2. ✅ 올바른 계정으로 로그인했나?
3. ✅ 접속 주소가 정확한가? (`http://`로 시작, `:3000` 포함)
4. ✅ MockupAI 서버가 가동 중인가? (관리자에게 문의)

### Q6. 다른 VPN과 함께 사용해도 되나요?

**A**: **권장하지 않습니다.** 
- 다른 VPN과 충돌할 수 있습니다
- Tailscale 사용 시에는 다른 VPN을 끄고 사용하세요

### Q7. 핸드폰에서도 사용할 수 있나요?

**A**: 네, 가능합니다!
- **iOS**: App Store에서 "Tailscale" 검색하여 설치
- **Android**: Google Play에서 "Tailscale" 검색하여 설치
- 같은 계정으로 로그인하면 됩니다

## 🔧 문제 해결

### Tailscale 아이콘이 보이지 않음

**Windows:**
1. 작업표시줄 오른쪽의 **↑ 화살표** 클릭
2. 숨겨진 아이콘 중 Tailscale 찾기
3. 없다면 시작 메뉴에서 "Tailscale" 검색하여 실행

**Mac:**
1. 응용 프로그램에서 Tailscale 실행
2. 상단 메뉴바 확인 (시계 근처)

### "연결 안 됨" 상태가 계속됨

1. Tailscale 아이콘 우클릭 → **"Log out"**
2. 다시 **"Log in"** 클릭
3. 계정 정보 다시 입력

### 접속 주소가 작동하지 않음

1. **주소 확인**: `http://`로 시작하는지, `:3000`이 포함되어 있는지
2. **Tailscale 상태 확인**: 아이콘이 연결됨 상태인지
3. **서버 상태 확인**: 관리자에게 서버가 정상 가동 중인지 문의

### Mac에서 "시스템 확장 프로그램" 오류

1. **시스템 설정** 열기
2. **개인정보 보호 및 보안** 선택
3. 하단의 **"허용"** 버튼 클릭
4. Tailscale 다시 실행

### 여전히 해결되지 않으면

관리자에게 다음 정보와 함께 문의하세요:
- 사용 중인 운영체제 (Windows 10/11, macOS 버전)
- Tailscale 연결 상태
- 정확히 어떤 오류 메시지가 나타나는지
