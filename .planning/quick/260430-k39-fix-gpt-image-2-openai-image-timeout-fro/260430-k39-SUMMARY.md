# Quick Task 260430-k39 Summary

OpenAI GPT Image 2 SDK timeout을 `OPENAI_IMAGE_TIMEOUT_MS`로 설정 가능하게 만들고 기본값을 `600_000`ms로 올렸습니다. SDK retry budget은 `maxRetries: 0`으로 유지했습니다. OpenAI IP Change와 Sketch-to-Real 품질 모드에는 사용자가 지정한 예상 대기시간을 표시합니다.

## Changes

- `apps/api/src/services/openai-image.service.ts`
  - `OPENAI_IMAGE_TIMEOUT_MS` resolver 추가
  - empty, non-numeric, zero, negative env 값은 `600_000`으로 fallback
  - 다섯 개 OpenAI client 생성 지점의 hardcoded `60_000` timeout 제거
- `apps/api/src/services/__tests__/openai-image.service.test.ts`
  - 기본 timeout `600_000` assertion 반영
