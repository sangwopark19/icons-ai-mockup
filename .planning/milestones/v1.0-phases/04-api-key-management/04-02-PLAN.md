**3. Add methods to AdminService class:**

- `listApiKeys()`: `prisma.apiKey.findMany({ orderBy: { createdAt: 'desc' }, select: { id, alias, maskedKey, isActive, callCount, lastUsedAt, createdAt } })` — explicitly select to EXCLUDE encryptedKey. Return typed as `ApiKeyListItem[]`.

- `createApiKey(alias: string, rawKey: string)`: Extract last 4 chars as maskedKey. Call `encrypt(rawKey, getEncryptionKey())`. `prisma.apiKey.create({ data: { alias, encryptedKey, maskedKey, isActive: false } })`. Return the created item as ApiKeyListItem (exclude encryptedKey from return).

- `deleteApiKey(id: string)`: `findUnique({ where: { id } })`. If `isActive === true`, throw `new Error('활성 키는 삭제할 수 없습니다')`. Otherwise `prisma.apiKey.delete({ where: { id } })`.

- `activateApiKey(id: string)`: Use `prisma.$transaction([ prisma.apiKey.updateMany({ where: {}, data: { isActive: false } }), prisma.apiKey.update({ where: { id }, data: { isActive: true } }) ])`. Return the updated key as ApiKeyListItem (fetch after transaction with select).

- `getActiveApiKey()`: `prisma.apiKey.findFirst({ where: { isActive: true } })`. If null, throw `new Error('Gemini API 키가 설정되지 않았습니다')`. Decrypt `encryptedKey` with `decrypt(encryptedKey, getEncryptionKey())`. Return `{ id, key: decryptedKey }`.

- `incrementCallCount(id: string)`: `prisma.apiKey.update({ where: { id }, data: { callCount: { increment: 1 }, lastUsedAt: new Date() } })`.
