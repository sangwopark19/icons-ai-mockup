# Plan 08-02: V2 Project Entry And IP Change Form

<objective>
Expose `IP 변경 v2` beside the existing `IP 변경 v1` entry from the same project context and add a v2 form route that submits OpenAI IP Change requests with preservation defaults, two-candidate output, and Korean quality mode labels.
</objective>

<threat_model>
## Threats

| ID | Severity | Threat | Mitigation |
|----|----------|--------|------------|
| T-08-05 | medium | Product UI leaks provider/model names that Phase 8 explicitly hides | Render only `v1`, `v2`, `IP 변경 v1`, and `IP 변경 v2`; keep provider/model values only in request payload. |
| T-08-06 | high | v2 form accidentally submits to Gemini by omitting provider fields | Submit exact `provider: "openai"` and `providerModel: "gpt-image-2"` in the v2 create body. |
| T-08-07 | medium | v1 route behavior changes while adding v2 | Keep `/projects/:id/ip-change` and its create payload Gemini-default unless only user-facing `v1` label is added. |
</threat_model>

    Create `apps/web/src/app/projects/[id]/ip-change/openai/page.tsx` by reusing the existing IP Change form flow and applying v2-specific behavior:
    - page title: `IP 변경 v2`
    - subtitle: `기존 제품의 구조와 시점을 유지하면서 새 캐릭터를 적용합니다.`
    - source uploader label: `원본 제품 이미지`
    - character uploader label: `새 캐릭터 이미지`
    - primary CTA: `v2 목업 생성하기`
    - required image error: `원본 제품 이미지와 새 캐릭터 이미지를 모두 업로드해주세요.`
    - user instructions helper: `보존 규칙 안에서 추가 요청을 반영합니다.`

    Set v2 defaults:
    - `preserveStructure` initial state `true`
    - `fixedViewpoint` initial state `true`
    - `fixedBackground` initial state `true`
    - `preserveHardware` initial state `false`
    - `transparentBg` initial state `false`
    - `removeShadows` initial state `false`

    Add a keyboard-operable three-option quality control labeled `품질 모드`:
    - `빠른모드` maps to `low`
    - `균형모드` maps to `medium` and is default
    - `퀄리티모드` maps to `high`

    Submit the create body with exact values:
    - `projectId`
    - `mode: 'ip_change'`
    - `provider: 'openai'`
    - `providerModel: 'gpt-image-2'`
    - `sourceImagePath`
    - `characterImagePath`
    - `options.outputCount: 2`
    - `options.quality` set to `low`, `medium`, or `high`
    - existing preservation options and user instructions

    On success, navigate to `/projects/${projectId}/generations/${data.data.id}`.
  </action>
