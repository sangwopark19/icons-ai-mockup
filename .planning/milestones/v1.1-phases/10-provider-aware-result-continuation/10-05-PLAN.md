| T-10-05-03 | Tampering | lineage mixing | mitigate | Tests assert OpenAI style-copy never calls Gemini `thoughtSignature` branch. |
| T-10-05-04 | Repudiation | OpenAI support metadata | mitigate | Save OpenAI request/response/image-call metadata via existing `updateOpenAIMetadata` path. |
| T-10-05-05 | Tampering | linkage fallback policy | mitigate | Fallback once only for recoverable missing/expired/invalid response linkage or 5xx; never fallback on auth/key/permission errors. |
