    3. **Helper `buildImageWhere`** (private method or local function):
       - Takes `{ email?, projectId?, startDate?, endDate? }`
       - Returns Prisma where clause for `generatedImage`:
         - `email` -> `generation: { is: { project: { is: { user: { is: { email: { contains, mode: 'insensitive' } } } } } } }`
         - `projectId` -> `generation: { is: { projectId } }`
         - `startDate` -> `createdAt: { gte: new Date(startDate) }`
         - `endDate` -> `createdAt: { lte: new Date(endDate) }`
       - Merge nested `generation.is` conditions when both email and projectId present

    4. **Implement `listGeneratedImages`** following listUsers pattern:
       - Build where via `buildImageWhere`
       - `prisma.generatedImage.findMany` with `include: { generation: { include: { project: { include: { user: { select: { email: true } } } } } } }`
       - Also `select: { name: true }` on project for `projectName`
       - Map results to flatten nested joins into `userEmail` and `projectName`

    5. **Implement `deleteGeneratedImage(imageId: string)`**:
       - `prisma.generatedImage.findUnique({ where: { id: imageId } })`
       - If not found, throw `new Error('Image not found')`
       - `await uploadService.deleteFile(image.filePath)` (silently handles missing files)
       - `if (image.thumbnailPath) await uploadService.deleteFile(image.thumbnailPath)`
       - `await prisma.generatedImage.delete({ where: { id: imageId } })`
       - CRITICAL: Do NOT delete Generation record (Pitfall 2 from RESEARCH.md)

    6. **Implement `countImages(params)`**:
       - Build where via `buildImageWhere(params)`
       - Return `prisma.generatedImage.count({ where })`

    7. **Implement `bulkDeleteImages(params)`**:
       - Build where via `buildImageWhere(params)`
       - `const images = await prisma.generatedImage.findMany({ where, select: { id: true, filePath: true, thumbnailPath: true } })`
       - Loop through images, call `uploadService.deleteFile` for each filePath and thumbnailPath
       - `await prisma.generatedImage.deleteMany({ where })`
       - Return `{ deletedCount: images.length }`
